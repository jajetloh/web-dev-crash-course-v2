This workshop is not a detailed guide on HTML, CSS and JavaScript, but an introduction to some of the tools beyond just these three. There are many, many, many good tutorials and classes out there on these basics - if you are interested in learning more, you need only Google!

However, I have some particular recommendations for those who are interested in learning more. My most important tip is to learn by practising. For most, this means finding ideas you are passionate about (and are also suitable for front-end/web development), and building them! Knowing the theory is well and good, but is not much if it you can't efficiently build things in practice. For more topics and resources to look at as well...

Name Description
Git/Version Control Version Control software like Git is essential for anyone working in software development - front-end or not. Version control allows you to freely experiment without the risk of losing previously working versions of code, and allows software to be built collaboratively between multiple people.
GitHub Pages You can use GitHub pages to host static web pages for free! And just because it's static doesn't mean it can't do cool things... (see here or here)
Asynchronous Programming A different paradigm of programming from what you may be used to, but extremely important in front-end development, where you are dealing with human interactions, requests to servers, and potentially many combinations of strange behaviours. A good start is to understand async/await (e.g. from here)
Chrome DevTools Get familiar with your browser's developer tools! For Google Chrome, this is Chrome DevTools, but other browser will have similar tools. Knowing how to use browser developer tools is essential for efficiently debugging, testing and even prototyping web front-ends.
A Complete Guide to Flexbox A comprehensive guide on how to use CSS Flex functionality - whether that's in plain CSS, or as part of a CSS framework. A solid understanding of how to use Flex will do wonders in helping you produce beautiful layouts.
Fireship (YouTube channel) Short-form video content on all things development, but with emphasis on front-end development and related topics. I honestly find out about a lot of new and interesting frameworks, libraries, tools etc. from this channel. For starters on JavaScript Frameworks, I recommend Angular, React (for the haters) and Vue.
